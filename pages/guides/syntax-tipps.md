# Syntax Tipps

## Ueberschriften

Tippe `# ` fuer H1, `## ` fuer H2, `### ` fuer H3.

## Listen

- `- ` startet Aufzaehlung
1. `1. ` startet Nummerierung

## Zitat und Code

- `> ` startet Blockquote
- `` ``` `` startet Codeblock
- `` `inline` `` erzeugt Inline-Code

## Textformatierung

- `**fett**`
- `*kursiv*`
- `‾‾durchgestrichen‾‾`
